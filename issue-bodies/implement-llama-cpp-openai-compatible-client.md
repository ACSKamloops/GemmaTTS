# Implement llama.cpp OpenAI-compatible client

## Summary
Implement a provider client for llama.cpp server using OpenAI-compatible chat completions.

## Acceptance criteria
- [ ] Supports non-streaming chat completion.
- [ ] Supports streaming text events.
- [ ] Supports JSON/schema-constrained generation when configured.
- [ ] Handles timeout, cancellation, and retry policy.
